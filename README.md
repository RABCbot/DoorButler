# DoorButler
Smart video doorbell<br/>

## Goals
Device should provide the following capabilities
- Smart voice screening<br/>
- Video streaming<br/>
- Led status indicator<br/>
- REST api<br/>
- Mqtt support maybe?<br/>
- Presence detection maybe?<br/>
- Two way audio<br/>
